# Terraria-Mod-Manager（模组管理器）

A mod manager for Terraria, making it easy to browse, install, and manage your Terraria mods.

## Features

- 🎮 Browse and discover Terraria mods
- 📦 One-click install and update
- 🔧 Manage mod configurations
- 🌍 Community-driven mod sharing
